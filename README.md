#Webpack Nunjucks Simple Boilerplate

##[ENGLISH]

**Download :**
```
git clone https://github.com/aligundogdu/webpack-nunjucks-simple-boilerplate.git <folder_name>
cd folder_name
npm install
```
**Development :**

```
npm run dev
```

**Build :**

```
npm run build:prod
```

## Features

* Webpack Support
* Sass Compile (Watch)
* Css minify and compress 
* Image optimize
* Javascript minify and compress
* Browsersync
 
## Include

* Bootstrap
* Fontawesome
* Jquery
* Slick Slider
